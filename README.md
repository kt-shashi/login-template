# LoginTemplateApp

</div>

<div align="left">
<br>

A template android app that handles signup/signin of users. 

<br>
<hr>
<br>

## Features⚡️

* Signup with Mail (Verification link sent to mail) & Gmail
* Signin (Gmail & Other mails)
* Create user profile (Profile Name & Image)
* Update user profile 
* Logout
